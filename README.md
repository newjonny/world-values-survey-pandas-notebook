# world-values-survey-pandas-notebook
Analysis of selected questions of the World Values Survey.

```analysis.ipynb``` contains the full analysis with three additional data sets:
- GDP per capita<sup>[1]</sup>
- Human Development Index (HDI)<sup>[2]</sup>
- New business registrations per 1,000 inhabitants<sup>[3]</sup>

## Important Aspects
Data is from different years: the interviews of the World Values Survey have been conducted in various years.
